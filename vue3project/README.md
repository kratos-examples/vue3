# vue3project

Vue 3 + TypeScript frontend demo for kratos-examples/vue3.

Connects to two Kratos backend services via gRPC-to-HTTP clients generated by [vue3kratos](https://github.com/yylego/kratos-vue3).

## Services

- **StudentService** (demo1kratos, port 18000)
- **ArticleService** (demo2kratos, port 28000)

## Setup

```bash
npm install
npm run dev
```

## Dependencies

- [@yylego/grpc-to-http](https://www.npmjs.com/package/@yylego/grpc-to-http) — gRPC-to-HTTP runtime
- [@protobuf-ts/grpcweb-transport](https://www.npmjs.com/package/@protobuf-ts/grpcweb-transport) — transport
